## Project Description

This assignment shows how JavaScript can be used to develop interactive web pages. The topics that were covered include; DOM manipulation of HTML objects, handling user events, form validation, and development of an interactive To-do list application. The main purpose of this assignment is to learn how JavaScript works with HTML objects and reacts to user events to make web applications interactive.

## Technologies Used
- HTML
- VS
- JavaScript


## Lessons Learned

- How to access and modify various elements of a web page using JavaScript.
- Add elements to a web page. 
- Delete elements in a web page.
- Make a web page react to user actions like button clicks and form entries.
- How to verify that the user entered the required data in a form.
- how to create an event listener for multiple elements and even future elements.
- Creating an interactive web page rather than a static one.
